## Features

- Ten-stage time attack where every target requires an exact shape
- Later puzzles require planned line clears before reaching the final shape
- EASY and HARD modes
- Separate online time leaderboards for each mode

## How to Play

- **Move**: Left / Right arrow or A
- **Soft drop**: Down arrow or S
- **Rotate**: Z / X / Up arrow
- **Hard drop**: Space
- **Hold**: C
- **Restart the current puzzle**: R
