## Features

- Six pitch types to choose from
- Repeating the same location or pitch makes it easier for the batter to connect
- Select a control style before starting

## How to Play

Choose a player name and control style on the home screen, then start the game.

1. Select a pitch type using the buttons or number keys 1-6.
2. Drag around the strike zone to choose a target.
3. Stop the moving gauge inside the white area with the PITCH button, Space, or Enter.

Press **R** to return to the title screen.
